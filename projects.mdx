## Documentation AI

Documentation AI revolutionizes technical writing by automating content generation while maintaining quality and accuracy. This project leverages machine learning to create comprehensive documentation from code analysis.

<Tabs>
  <Tab title="Features" icon="star">
    <ExpandableGroup>
      <Expandable title="Automated Generation" default-open="true">
        Generate complete documentation sets from source code with minimal manual input.
      </Expandable>
      <Expandable title="Component Integration" default-open="false">
        Seamlessly integrate with existing documentation platforms using flexible APIs.
      </Expandable>
    </ExpandableGroup>
  </Tab>
  <Tab title="Implementation" icon="code">
    <CodeGroup tabs="Python,JavaScript">
      ```python
from doc_ai import Generator

generator = Generator(api_key='your_key')
docs = generator.generate_from_code('project.py')
```
      ```javascript
import { DocGenerator } from 'doc-ai';

const generator = new DocGenerator({ apiKey: 'your_key' });
const docs = await generator.generate('project.js');
```
    </CodeGroup>
  </Tab>
</Tabs>
